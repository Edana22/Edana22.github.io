Markdown 快速入门

约 723 字大约 2 分钟

2025-06-25

Markdown 是什么

  • Markdown 是一个 标记语言 ，用键盘输入标记对文本进行格式化。
  • Markdown 文件的后缀名是 .md （可以把文本文档的 .txt 直接修改后缀名为 .md）。
  • 不同软件支持的 Markdown 语法不同，各个软件差异很大。
  • 这些不同的语法可以称为 Markdown 的 方言 。由于 方言 差异过大，所以在本文中我们只介绍 Markdown 最基础的语法。
  • Markdown 的用键盘操作和 word 的快捷键性质不同。

相关信息

更多介绍查看官网： Markdown

提示

我们后面要学的 HTML 同样是 标记语言 。

因为 Markdown 的标记比 HTML 少，所以我们用 Markdown 入门前端 而不是 HTML 。

标记语言是什么

用某些约定好的符号，指定某个特殊的格式。

比如我们熟悉的 1. 表示序号 1， 2. 表示序号 2

用什么写 Markdown 文件

支持 Markdown 的编辑器很多，如 Tyopra、 notepad++、 vscode，这里我用的是 vscode。

Markdown 官网也列举了一些软件，可以到 编辑器 查看。

提示

如果你暂时不想下载编译器，也可以尝试 在线编辑器

Markdown 基本语法

  • 标题（Heading） # H1 ## H2 ### H3
  • 粗体（Bold） **bold text**
  • 斜体（Italic） _italicized text_
  • 引用块（Blockquote） > blockquote
  • 有序列表（Ordered List） 1. First item 2. Second item 3. Third item
  • 无序列表（Unordered List） - First item
  • 代码（Code） code
  • 分隔线（Horizontal Rule） ---
  • 链接（Link） [this is a link](https://www.example.com)
  • 图片（Image） ![alt text](cherry.png)
